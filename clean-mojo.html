testdb:clean

Full name:

io.metaloom.maven:testdb-maven-plugin:0.1.1:clean

Description:

The stop all containers that were previously started and delete the test provider config file.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.
  • Binds by default to the lifecycle phase: pre-clean.

Optional Parameters

Name Type Since Description

The settings can be used to configure the started postgresql container or for the use of an external database connection.
